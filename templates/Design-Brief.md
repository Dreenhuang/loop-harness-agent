## 3. 色彩系统

<!-- 填写指引：定义主色、辅助色、语义色和中性色，提供 HEX/RGB/HSL 值和使用规则。 -->

### 3.1 主色

| 色彩名称 | HEX | RGB | 用途 | 使用比例 |
|----------|-----|-----|------|----------|
| Primary | # | | 品牌/主操作/关键信息 | ~60% |
| Secondary | # | | 辅助操作/次要信息 | ~30% |
| Accent | # | | 强调/引导注意力 | ~10% |

### 3.2 语义色

| 语义 | HEX | 用途 |
|------|-----|------|
| Success | # | 成功状态/确认操作 |
| Warning | # | 警告/需注意 |
| Error | # | 错误/危险/删除 |
| Info | # | 提示/信息 |

### 3.3 中性色

| 色彩名称 | HEX | 用途 |
|----------|-----|------|
| Text-Primary | # | 主文本 |
| Text-Secondary | # | 次要文本 |
| Text-Disabled | # | 禁用文本 |
| Border | # | 边框/分割线 |
| Background-Primary | # | 主背景 |
| Background-Secondary | # | 次要背景/卡片 |
| Background-Elevated | # | 浮层/弹窗背景 |

### 3.4 色彩使用规则

- 主色不得用于大面积背景（≤ 30% 视觉面积）
- 语义色仅用于对应语义场景，禁止混用
- 文本与背景对比度 ≥ 4.5:1（WCAG AA）
- 深色模式需单独定义色板

### 4.2 字号层级

| 层级 | Token 名称 | 字号 | 字重 | 行高 | 用途 |
|------|-----------|------|------|------|------|
| Display | --font-display | 48px | 700 | 1.2 | 首屏大标题 |
| H1 | --font-h1 | 36px | 700 | 1.3 | 页面主标题 |
| H2 | --font-h2 | 28px | 600 | 1.3 | 章节标题 |

| Caption | --font-caption | 12px | 400 | 1.4 | 标注/说明 |
| Overline | --font-overline | 12px | 500 | 1.2 | 标签/分类 |

### 4.3 字体使用规则

- 正文最小字号不低于 12px
- 标题层级不得跳级（H1 → H3 禁止）
- 中英文混排时英文使用对应字体族
- 行高统一使用倍数制（1.2 / 1.3 / 1.4 / 1.5 / 1.6）

## 5. 间距系统

<!-- 填写指引：基于 4px/8px 网格定义间距 Token，确保布局一致性。 -->

### 5.1 间距 Token

| Token 名称 | 值 | 用途 |
|-----------|-----|------|
| --space-1 | 4px | 图标与文字间距 |
| --space-2 | 8px | 紧凑元素间距 |
| --space-3 | 12px | 同组元素间距 |
| --space-4 | 16px | 组件内间距 |
| --space-5 | 24px | 组件间间距 |
| --space-6 | 32px | 区块间间距 |
| --space-7 | 48px | 大区块间距 |
| --space-8 | 64px | 页面级间距 |

### 5.2 间距使用规则

- 所有间距必须使用 Token 值，禁止硬编码
- 组件内部使用 --space-1 ~ --space-4
- 组件之间使用 --space-4 ~ --space-6
- 页面区块之间使用 --space-6 ~ --space-8
- 间距方向：优先使用 gap，其次 margin-bottom

## 6. 组件风格

<!-- 填写指引：定义通用组件的视觉风格，包括圆角、阴影、边框、动效。 -->

### 6.1 圆角

| Token 名称 | 值 | 用途 |
|-----------|-----|------|
| --radius-sm | 4px | 小元素（Tag/Badge） |
| --radius-md | 8px | 中等元素（Button/Input） |
| --radius-lg | 12px | 大元素（Card/Modal） |
| --radius-xl | 16px | 特大元素（Sheet/Drawer） |
| --radius-full | 9999px | 圆形（Avatar） |

### 6.2 阴影

| Token 名称 | 值 | 用途 |
|-----------|-----|------|
| --shadow-sm | 0 1px 2px rgba(0,0,0,0.05) | 轻微浮起 |
| --shadow-md | 0 4px 6px rgba(0,0,0,0.1) | 卡片 |
| --shadow-lg | 0 10px 15px rgba(0,0,0,0.1) | 弹窗 |
| --shadow-xl | 0 20px 25px rgba(0,0,0,0.15) | 全屏浮层 |

### 6.3 边框

| Token 名称 | 值 | 用途 |
|-----------|-----|------|
| --border-light | 1px solid var(--border) | 默认边框 |
| --border-focus | 2px solid var(--primary) | 聚焦边框 |
| --border-error | 2px solid var(--error) | 错误边框 |

### 6.4 动效

| 类型 | 时长 | 缓动函数 | 用途 |
|------|------|----------|------|
| Micro | 100ms | ease-out | 按钮按压/开关切换 |
| Small | 200ms | ease-in-out | 下拉展开/Tooltip |
| Medium | 300ms | ease-in-out | 模态框/抽屉 |
| Large | 500ms | ease-in-out | 页面切换 |
