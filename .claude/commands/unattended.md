# /unattended - 无人值守模式

激活无人值守模式，6 条铁律：
1. **不中断原则**：除非触发"用户确认边界"，否则 AI 自主决策
2. **最小影响原则**：异常时优先回滚或降级
3. **完整执行原则**：宁可慢也要完成，不留半成品
4. **决策可审计**：所有自动决策记录到黑板
5. **时间有预算**：到点强制保存并报告
6. **早晨有报告**：用户醒来看到完整的"夜间作业报告"

请执行：

1. 提取时间预算参数（如 "时间预算 8 小时"）
2. 调用 `mcp__loop-agent__start_loop` 传入：
   - `mode`: "unattended"
   - `time_budget_hours`: 解析后的数字
3. 激活 `unattended-mode` Skill
4. 输出："✅ 无人值守模式已激活，⏰ 时间预算：X 小时"

**时间预算解析**（见 `CLAUDE.md` §八）：
- "时间预算 X 小时" → X
- "今晚完成" / "明天早上" → 9 小时
- "通宵跑" → 8 小时
- 默认 → 9 小时
