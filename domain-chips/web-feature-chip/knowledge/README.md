# Domain Chip Knowledge Base

> 知识库目录，存储 Web Feature 开发领域的经验知识

## 目录结构

- `problems/` - 问题解决方案
- `architectures/` - 架构决策记录
- `components/` - 可复用组件
- `best-practices/` - 最佳实践
- `pitfalls/` - 常见陷阱

## 用途

- 供 @Knowledge-Curator 沉淀融合执行中的经验模板与反模式
- 供 @Architect 查询领域架构决策
- 供 @Backend/@Fullstack-Coder 查询可复用组件和最佳实践
- 供 autoloop 自进化循环基于 benchmark 改进
