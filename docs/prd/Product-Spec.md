## 1. 产品概述

AI 任务调度平台 是一个基于 Loop-Harness-Agent 开发的项目。

## 2. 核心功能

- 多 Agent 并发
- 任务编排
- 进度跟踪

## 3. 用户故事

暂无用户故事

## 4. 验收标准

- [ ] 所有核心功能可正常使用
- [ ] 通过全部自动化测试
- [ ] 性能指标达标（P95 < 300ms）
- [ ] 安全审计通过

## 5. 技术约束

- 前端框架: React 18 + TypeScript
- 后端框架: Node.js + Express/Fastify
- 数据库: PostgreSQL / SQLite
- 部署方式: Docker + Nginx
