# Loop 工程成熟度自评工具

> **版本**: v1.0
> **生效日期**: 2026-06-15
> **解决偏差**: D-12（无成熟度自评工具）
> **蓝皮书**: 第十章 5 级路径

## 一、5 级成熟度

| 级别 | 名称 | 特征 | 典型用户 |
|------|------|------|----------|
| L0 | 探索者 | 手动 prompt 阶段 | 初学者 |
| L1 | 单 Loop 用户 | `/loop` `/goal` 监控单一任务 | 个人开发者 |
| L2 | 验证者 | 加 4 道门禁 + 3 道硬刹车 | 严谨开发者 |
| L3 | 资产化者 | 沉淀 Skills + Routines 持久化 | 团队 Leader |
| L4 | 编排者 | 多 Agent Dynamic Workflows | 架构师 |
| L5 | 自进化者 | Autoloop + Benchmark 驱动 | 顶级组织 |

## 二、自评 30 题

### L1 阶段（6 题）
- [ ] 1. 你会用 `/loop` 或 `/loop-agent` 命令吗？
- [ ] 2. 你能用自然语言触发 Loop Agent 吗？
- [ ] 3. 你理解 Stop Condition（停止条件）吗？
- [ ] 4. 你能设置 max_turns 上限吗？
- [ ] 5. 你能读 blackboard 中的状态吗？
- [ ] 6. 你能恢复失败的 Loop（resume）吗？

### L2 阶段（6 题）
- [ ] 7. 你有 4 道门禁 Skills 吗？
- [ ] 8. 你设了 max_iterations / max_budget_usd 吗？
- [ ] 9. 你启用了 no_progress_detection 吗？
- [ ] 10. 你有独立的 Maker-Checker 吗？
- [ ] 11. 你的 Agent 有显式 output contract 吗？
- [ ] 12. 你的决策有记录到 decision_log.json 吗？

### L3 阶段（6 题）
- [ ] 13. 你有 ≥ 10 个可复用 Skills 吗？
- [ ] 14. 你用 Routines 或 Schedule 持久化任务吗？
- [ ] 15. 你的失败案例有文档化吗？
- [ ] 16. 你的知识库条目 ≥ 20 个吗？
- [ ] 17. 你有 Worktree 隔离多 Agent 吗？
- [ ] 18. 你的 Skills 有版本管理吗？

### L4 阶段（6 题）
- [ ] 19. 你有完整的 Workflow Blueprint 吗？
- [ ] 20. 你能并行扇出多 Agent 吗？
- [ ] 21. 你能 Map-Reduce 编排吗？
- [ ] 22. 你有辩论对抗模式吗？
- [ ] 23. 你的 Orchestrator 只路由不推理吗？
- [ ] 24. 你有 Domain Chip 吗？

### L5 阶段（6 题）
- [ ] 25. 你有 Autoloop 自进化能力吗？
- [ ] 26. 你有 Benchmark 评估集吗？
- [ ] 27. 你能变异 + 评分 + 优胜劣汰吗？
- [ ] 28. 你有 Eval Gate 防止回归吗？
- [ ] 29. 你有跨项目 Skill 共享机制吗？
- [ ] 30. 你能"写 Loop 一次，跨项目复用"吗？
