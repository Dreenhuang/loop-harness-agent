**发布时间**: 2026-06-21（预计 2 天开发）

| F006 | WebSocket 实时连接 | 后端 WS 服务 + 前端订阅 | P0 | 4h |

## M2: 指标体系与移动端优化 (v1.1.0)

**目标**: 提升数据深度，覆盖移动场景

**发布时间**: M1 上线后 1 周

### 新增功能（P1）

| 功能 ID | 功能名称 | 描述 | 优先级 | 工作量估算 |
|---------|---------|------|--------|-----------|
| F007 | 异常告警面板 | 偏离检测、Token 超预算等告警 | P1 | 6h |
| F008 | 历史数据回放 | 时间轴回放 Agent 运行历史 | P1 | 8h |
| F009 | 多视图切换 | 总览/详情/日志/指标四种模式 | P1 | 4h |
| F010 | 响应式布局完善 | 平板 + 手机完整适配 | P1 | 8h |

### 技术改进
- 引入 ECharts 图表库，增强数据可视化
- 日志虚拟滚动优化（处理万级日志量）
- WebSocket 连接池管理（支持多客户端）
- SQLite 数据库索引优化（查询性能提升）

### 验收标准
- ✅ 告警延迟 < 3秒
- ✅ 历史回放流畅（快进/暂停/拖动）
- ✅ 移动端单手可操作
- ✅ 10,000 条日志渲染 < 100ms

## M3: 成熟期 - 告警系统与多租户 (v2.0.0)

**目标**: 生产级稳定性，支持多项目并行监控

**发布时间**: M2 上线后 2 周

### 新增功能（P2 + 增强）

| 功能 ID | 功能名称 | 描述 | 优先级 | 工作量估算 |
|---------|---------|------|--------|-----------|
| F011 | 深色模式 | 明暗主题一键切换 | P2 | 4h |
| F012 | 导出报告 | PDF/JSON 格式导出 | P2 | 4h |
| F013 | 多项目管理 | 支持同时监控多个 Loop-Agent 项目 | P2 | 12h |
| F014 | Webhook 通知 | 告警推送到钉钉/飞书/Slack | P2 | 6h |
| F015 | 自定义仪表盘 | 用户自定义面板布局和组件 | P2 | 16h |

### 架构升级
- 引入 Redis 缓存层（多实例共享状态）
- PostgreSQL 替代 SQLite（支持并发写入）
- Docker 容器化部署（一键启动）
- Kubernetes 编排文件（生产环境）
- Prometheus + Grafana 监控集成

### AI 能力集成（预留）
- NLP 日志智能摘要
- Token 消耗趋势预测
- 资源分配策略推荐
- 异常根因自动分析

### 验收标准
- ✅ 支持 ≥ 5 个项目并行监控
- ✅ 并发用户数 ≥ 100
- ✅ 可用性 99.9%
- ✅ 容器化部署，启动时间 < 30秒

## 附录：版本号规则

采用语义化版本控制 (SemVer)：

```
MAJOR.MINOR.PATCH

MAJOR: 不兼容的 API 变更
MINOR: 向后兼容的功能新增
PATCH: 向后兼容的问题修复
```

示例：
- `v1.0.0` → M1 MVP 首次发布
- `v1.0.1` → M1 Bug 修复
- `v1.1.0` → M2 新增功能
- `v2.0.0` → M3 架构升级（多租户）
